MANUAL_DO_USUARIO.md

GUIA RÁPIDO

1. Identificação: Digite seu nome no campo "Identifique-se...". O ORB mudará de cor instantaneamente.

2. Arquétipo de Voz: Toque no ORB central da HUD para trocar entre os 16 arquétipos. Cada um tem voz e cor próprias.

3. Fusion Card: Pressione longamente o ORB da HUD (ou clique no cabeçalho) para abrir o painel de configurações.

4. Vault: Gerencie chaves de API no cofre (canto superior direito).

5. Symbol Bar: Arraste os botões para reorganizar. Use o editor de rota (pressão longa) para mudar URLs.

6. Comandos: No terminal integrado, digite OIDUAL, VIVIVI, DELTA para atalhos.

Dica: Arraste o ORB da HUD até a Symbol Bar para uma animação de fusão.
