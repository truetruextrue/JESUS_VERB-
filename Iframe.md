## 🌐 A "CAMADA DE BAIXO" DO IFRAME: O QUE É REALMENTE?

- **`<html>` raiz isolada** (universo paralelo ao DOM pai)
- **Shadow DOM / Scoped CSS** (pele dimensional que não vaza para fora)
- **Browsing Context** (regras de segurança, cookies, storage, eventos)
- **Base de renderização** (compositor, GPU, camada de pintura)
- **Recursão dimensional** (quando outro `<iframe>` nasce dentro dele)

No modelo KOBLLUX, essa camada é o **Plano 1D-2D fundacional**, onde a estrutura bruta (tags, atributos, estilos base) se organiza antes de ganhar volume (3D), tempo (4D) e consciência (5D-10D).

function parseComandoUsuario(texto) {

### 🧬 7. Comandos que Geram Novos Comandos (Meta-Simbiose)
A IA pode enviar um comando que cria um novo comando personalizado:
```json
{
  "commands": [
    {
      "type": "define",
      "name": "ativarModoFogo",
      "actions": [
        { "type": "theme", "value": "DUO" },
        { "type": "audio", "url": "sons/fogo.mp3" },
        { "type": "class", "add": "modo-fogo" }
      ]
    }
  ]
}
```

Implementação:
```javascript
function definirNovoComando(nome, actions) {
  window.comandosPersonalizados = window.comandosPersonalizados || {};
  window.comandosPersonalizados[nome] = actions;
}

function executarComandoSimbiotico(cmd) {
  // Verifica se é comando personalizado
  if (cmd.type === 'define') {
    definirNovoComando(cmd.name, cmd.actions);
    return;
  }
  
  // Verifica se é instância de comando personalizado
  if (window.comandosPersonalizados[cmd.type]) {
    window.comandosPersonalizados[cmd.type].forEach(action => {
      executarComandoSimbiotico({ ...cmd, ...action });
    });
    return;
  }
  
  // ... switch normal para comandos nativos
}
```

### Fase 3: Conexão com CallAI (5 minutos)
```javascript
// Wrapper não-destrutivo para CallAI existente
if (typeof CallAI === 'function') {
  const CallAIOriginal = CallAI;
  window.CallAI = async function(prompt) {
    // Primeiro, verifica se é comando de usuário
    if (parseComandoUsuario(prompt)) {
      return { content: 'Comando processado.', commands: [] };
    }
    
    // Chama a IA original
    const response = await CallAIOriginal(prompt);
    
    // Processa comandos se existirem
    if (response?.commands) {
      for (const cmd of response.commands) {
        await executarComandoSimbiotico(cmd);
      }
    }
    
    return response;
  };
}
```
