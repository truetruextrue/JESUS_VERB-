makeOrbAvatar(name, size = 36)
Gera SVG determinístico (seed = soma charCodes) com dois stops HSL. Renderiza orb em 32×32 viewBox · 3 círculos (glow / core / ring).
trigger: di_syncNameUI()
di_syncNameUI(name)
Single source of truth para o nome. Persiste em localStorage.di_userName + userName, propaga em #lblName · #actName · #smallText e regenera os três orbs.
trigger: #inputUser input · #infodoseNameInput input · boot
toggleSection(id)
Accordion exclusivo · fecha todos .activation-card e reabre o alvo se estava oculto.
trigger: .activation-toggle onclick
setMode(mode)
Troca o botão .active-mode entre #btnModeCard / Orb / Hud. Loga apenas — mudança de modo é tratada externamente.
trigger: .mode-btn onclick
toggleCard()
Alterna .closed no #mainCard. Quando reabre, foca #inputUser após 300ms.
trigger: #smallPreview click
updateClock()
Atualiza #clockTime em pt-BR HH:MM · loop setInterval 1000ms.
trigger: boot + interval
openModal(id) · closeModal(id)
Toggle de .active em .modal-overlay. Stateless — sem trap de foco / scroll-lock.
trigger: #avatarTarget · #closeKeysBtn · #lockVaultBtn · #vault*Btn
.drag-handle mousedown
Handler inline · habilita drag absoluto do #mainCard via offsets. Sem touch · sem snap · sem clamp de viewport.
trigger: pointer drag
