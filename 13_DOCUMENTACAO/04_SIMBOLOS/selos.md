## O Que É um Selo KOBLLUX

Um **selo** é a assinatura que prova que um artefato foi **gerado com intenção, verificado e publicado**. No KOBLLUX, o selo é sempre `∆7` — o sétimo delta, a síntese dos sete selos da criação.

```
∆7 = SELAR = 0×07 = 777Hz = JSON-LD = ✧⃝⚝
```

## Como Aplicar o Selo ∆7

Todo artefato criado no sistema KOBLLUX deve ser selado. O processo:

**1. Em código Python:**
```python
import hashlib
from datetime import datetime

def selo_d7(conteudo: str) -> str:
    h = hashlib.sha256(conteudo.encode()).hexdigest()[:16]
    ts = datetime.utcnow().isoformat() + "Z"
    return f"∆7 · {h} · {ts}"
```

**2. Em git (commit):**
```bash
git commit -m "CADIAL · Descrição do artefato · ∆7"
git push origin main
# O push É o selo — publicar é selar
```

**3. Em JSON (manifest/profile):**
